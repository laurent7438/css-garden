About the project

Learn CCS Garden is a project by Ammad Shadeed@shadeed9 I made this project to practise and get the best out of CSS. Through our work as Front End Developers, we might need to do variations of a specific component or page. It's better to do the change in CSS only, without changing the HTML markup, unless it's impossible to do the design change without editing the HTML. The goal of this project is to challenge ourselves to do as much variation as possible, without changing the HTML markup for the design problem we're working on. Are you in?
